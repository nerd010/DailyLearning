16. 区块链项目对比
```table
 | * | Hyperledger Fabric | Bitcode | Ethereum |
| 项目定位 | 通用联盟链平台 | 数字货币系统 | 通用公有链平台 |
管理方式 | Linux 基金会 | 社区  | 社区 （众筹）|
| 货币 | 无 | BTC 比特币 | Ether以太币 |
| 挖矿 | 无 | 有  | 有 | 
| 状态数据方式 | 键值数据、文档数据 | 交易数据 | 账号数据 |
| 共识网络 | PBFT,  SBFT 等 | PoW, PoS | 
| 网络 | 公开或私有 | 公开 | 公开 | 
| 隐私性 | 有 | 无 | 无 |
| 智能合约 | Go, Java 等多种语言 | 无 | Solidity |
```
